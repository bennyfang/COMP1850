Today I will cover two new tags that we have not covered in class as shown below:

The tag: The <button> tag defines a clickable button. Inside a <button> element you can put content, like text or images. Button attributes could specify things such as font size, button size, color and action taken after button click (new link, javascript action, etc.). The example below showcase change in size, text color, background color as well as a JavaScript action that we will learn later in the course!

The nav tag: The <nav> tag (introduced in HTML5) defines a set of navigation links. It is intended only for major block of navigation links. Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content. Styling, besides customizing color, font, etc., could also specify how you would like to list the link to diplay when you have, for example, <ul> and <li> tags inside it.
