10 Useful Resources for Web Development:

# Link Description
1 W3School HTML Tutorial A free online tutorial for HTML (version HTML5). Also includes resources and references to HTML tags and attributes
2 W3School CSS Tutorial A free online tutorial for CSS and CSS3. Also includes resources and references to CSS properties
3 W3C Markup Validation Service A free online tool to validate your HTML and markup files
4 W3C CSS Validation Service A free online tool to validate your CSS files
5 BCIT COMP1850 Homepage A BCIT course website about introduction to web development, including HTML, CSS and JavaScript
6 CSS Zen Garden A site that demonstrate of what can be accomplished through CSS-based design. Includes several themes written in basic HTML, CSS and JS that are downloadable
7 Google Fonts A Google-hosted site that allows you to use its 800+ web-friendly font families for free
8 Bootstrap A front-end framework for building responsive websites. It makes your websites responsive and look good both on small and on huge screens with a well done grid system and tons of CSS and JS features
9 Codepen A platform for showcasing impressive CSS3 and JS demos. Whether you are on the hunt for cool buttons or simply looking for some inspiration, do check CodePen out
10 Smashing Magazine A well-known news site not to be overlooked. With new articles every other day it offers you great resources about HTML, CSS, JavaScript-related topics.
